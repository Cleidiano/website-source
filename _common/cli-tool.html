Install Command line tool

```text ## Linux install curl -s https://ebean.io/cli-dist/install | bash ```

The command line tool is optional, we don't strictly need it but it does some useful tasks. We can do all these tasks manually.

  • Generates Finders
  • Generates ebean.mf
  • Add GenerateDbMigration (code to generate DB Migrations)
  • Add application-test.yml
  • Add logback-test.xml

Alternate manual installation (Linux)

```text cd ~/.local/bin curl https://ebean.io/cli-dist/ebean-init-1.9.jar > ebean-init-1.9.jar curl https://ebean.io/cli-dist/ebeaninit > ebeaninit ln -s ebean-init-1.9.jar ebean-init.jar chmod +x ebeaninit ```
