DatabaseConfig

io.ebean.DatabaseConfig has all the configuration available to then create a a Database instance.

```java DatabaseConfig config = new DatabaseConfig(); // set configuration options config.set... ... // and/or load configuration properties file config.loadFromProperties(); // create the Database instance Database database = DatabaseFactory.create(config); ``` <@next_edit "DatabaseFactory" "databasefactory" "/docs/introduction/databaseconfig.html"/>
