<#assign Encrypted = "true">

@Encrypted

Annotate properties that will be store in encrypted form in the database.

Example

  @Entity
  @Table(name="patient")
  public class Patient {

    ...
    // database side encryption
    @Encrypted
    String name;

    // database side encryption
    @Encrypted
    LocalDate dob;

    // client side encryption
    @Lob
    @Encrypted(dbEncryption=false)
    String description;

  }
