Database (EbeanServer)

io.ebean.Database provides the main API including:

  • Persisting - save, insert, update, delete
  • Transactions
  • Queries

A io.ebean.Database instance has an JDBC DataSource associating it one to one to an actual database.

Note that a io.ebean.Database can have a second read only DataSource that can be used for performance reasons (for read only queries) and potentially connect to database read replicas.

Default database

One database can be specified as the default database. The default io.ebean.Database instance is registered with DB and then can be obtained via DB.getDefault().

Named database

Each io.ebean.Database has a name and can be registered with DB and then obtained by its name via DB.byName()
