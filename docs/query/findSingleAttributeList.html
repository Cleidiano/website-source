findSingleAttributeList

Execute the query returning a list of values for a single property. We need to use the select() method to select which column you want to retrieve the value

Example 1:

```java List<|String> names = new QCustomer() .select("name") .order().name.asc() .findSingleAttributeList(); ```

Example 2:

```java List<|String> names = new QCustomer() .setDistinct(true) .select("name") .status.equalTo(Customer.Status.NEW) .orderBy().name.asc() .setMaxRows(100) .findSingleAttributeList(); ```
