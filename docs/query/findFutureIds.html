findFutureIds

Similar to findIds you can find a list of Ids in a background thread using findFutureIds.

This returns a Future object which can be used to cancel, check the execution status (isDone etc) and get the value (with or without a timeout).

```java FutureIds<|Long> futureIds = futureList = Order.find.where() .status.equalTo(Order.Status.NEW) .findFutureIds(); ``` And to get the actual computed value ```java // Waits (blocks) for the calculation to be finished and returns the value List<|Long> ids = futureIds.get(); // Waits for the calculation to be finished and returns the value // if it happens under 10 seconds, otherwise throw an exception List<|Long> ids = futureIds.get(10, TimeUnit.SECONDS); if (futureIds.isDone()) { // Here we can fetch the value (assuming that everything went ok) } ```
