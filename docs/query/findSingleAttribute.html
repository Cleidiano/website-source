findSingleAttribute

Execute a query retrieving the value of a single column.

To retrieve the value from a single property of a specific model, we use findSingleAttribute. We need to use the select() method to select which column to retrieve the value for. 

 String name = new QCustomer()
     .select("name")
     .id.equalTo(42)
     .findSingleAttribute();

This translates to an SQL query that only has the single property in the select clause, like the following: 

    select t0.name from customers t0 where t0.id = 42

If we use more than one value in the select clause (like select("name,email") only the first will be returned.

findSingleAttributeList

Execute the query returning a list of values for a single property. We need to use the select() method to select which column you want to retrieve the value

Example 1:

List<|String> names = new QCustomer()
    .select("name")
    .order().name.asc()
    .findSingleAttributeList();

Example 2:

List<|String> names = new QCustomer()
    .setDistinct(true)
    .select("name")
    .status.equalTo(Customer.Status.NEW)
    .orderBy().name.asc()
    .setMaxRows(100)
    .findSingleAttributeList();
