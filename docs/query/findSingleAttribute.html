findSingleAttribute (retrieve the value of a single column for a single model)

If you want to retrieve the value from a single property of a specific model, you can use the findSingleAttribute method. You need to use the select() method to select which column you want to retrieve the value ```java String name = DB.find(Customer.class) .select("name") .where() .idEq(42) .findSingleAttribute(); ``` This translates to an SQL query that only has the single property in the select clause, like the following: ```sql select t0.name from customers t0 where t0.id = 42 ``` If you use more than one value in the select clause (like select("name,email"), only the first will be returned.
